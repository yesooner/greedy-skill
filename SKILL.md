## 🔥 十大核心创新（正经解说版）

### 💡 创新 1: 多尺度时间窗口

> 同时盯着短期、中期、长期记忆，拒绝"健忘症"

```
短记忆 (10轮): weight=0.5, alpha=0.9  ← 最近发生啥了？
中记忆 (50轮): weight=0.3, alpha=0.7  ← 中期趋势咋样？
长记忆 (200轮): weight=0.2, alpha=0.5 ← 长期规律是啥？

多尺度置信度 = Σ(窗口置信度 × 窗口权重)
```

### 🧠 创新 2: 元学习 (MAML风格)

> 学霸不是天生，是掌握了"怎么学习"的能力！

```
Fast权重: 快速适应当前任务 ← 短期记忆
Slow权重: 跨任务泛化知识   ← 长期积累

元学习率: 0.01
每20轮从经验回放采样更新一次
```

### 🎯 创新 3: 注意力机制策略路由

> 7 种策略同台竞技，注意力机制选出最适合的那个！

```
Query向量: 8维随机向量（当前的"问题"）
Key/Value向量: 每个策略的"简历"
点积注意力: attention = softmax(Q·K/√d)·V

7策略:
  conservative → 保守派
  aggressive   → 激进派
  exploratory  → 探索家
  adaptive     → 万金油
  balanced     → 平衡大师
  random_walk  → 随机游走
  meta         → 元策略（终极BOSS）
```

### 📊 创新 4: 层次模式图 (HMG)

> 先分类，再推理，最后决策，三层递进

```
Level 0: action | decision | state_change | outcome | cascade
         ↓          ↓           ↓             ↓         ↓
Level 1: goal_oriented | reactive | deliberative
         ↓               ↓            ↓
Level 2: exploration | exploitation
```

### 📈 创新 5: 协方差自适应 (CMA-ES启发)

> 自适应调参，让置信度/动量/熵自动找到最优平衡

```
3x3协方差矩阵: 置信度/动量/熵 三个维度
均值向量: {confidence:0.7, momentum:0.5, entropy:0.5}
学习率: 0.1 ← 不要学太快，会震荡！
```

### ⚔️ 创新 6: 对抗性自适应

> 没有对手怎么变强？给自己加个"噪声敌人"练练！

```
对抗噪声: ε=0.05
噪声更新: noise = 0.9*noise + 0.1*new_noise
置信度扰动: confidence * (1 + noise)

越挫越勇，越扰越稳 💪
```

### 🔄 创新 7: 经验回放

> 错题本是学霸的标配，AI 也不例外！

```
缓冲区大小: 100条 ← 装得下过去的教训
批采样: 10条/次   ← 每次抽检10道错题
采样频率: 每20轮   ← 定期复盘
元学习数据来源    ← 从错题中学习新知识
```

### 🏆 创新 8: 多目标优化 (Pareto)

> 我全都要！Oracle 通过率、熵、置信度一个都不能少！

```
目标1: Oracle通过率 (target=75%, weight=0.4) ← 做对题！
目标2: 熵 (target=50%, weight=0.3)           ← 不要太躺平
目标3: 置信度 (target=70%, weight=0.3)        ← 要有底气

Pareto前沿: 最多20个非支配解 ← 最优解集合
```

### 🎲 创新 9: 连续动作空间

> 策略选择不是非黑即白，而是连续渐变的灰色地带！

```
策略选择时考虑:
- UCB1置信度 ← 历史战绩
- 熵奖励     ← 探索动力
- 混沌扰动   ← 偶尔疯狂一下
- 注意力分数 ← 实时评估
- 探索奖励   ← sqrt(2*log(t)/N)
```

### 🔮 创新 10: 层次推理

> 先定性，再定位，最后定策略——三层推理缺一不可！

```
输入: 基础模式 (action/decision/...)
推理: L0 → L1 → L2 三层映射
输出: 层次化预测 (level0 + level1 + level2)
      好比医生问诊：症状→病因→处方
```

## 🚀 使用方法

```bash
# 运行 Greedy-skill V300
node skills/Greedy-skill/scripts/greedy-loop.js --cycles 400 --interval 50 --flavor alibaba
```

**可选参数：**
- `--cycles 400` — 跑 400 轮（默认 200）
- `--interval 50` — 每 50 轮输出一次状态（默认 10）
- `--flavor alibaba` — 选择大厂味道（默认阿里味 🟠）
